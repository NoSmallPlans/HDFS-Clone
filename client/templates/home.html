Seattle University File System

A cloud based file management system.

SUFS Features:


Download File

- download a file given name from SUFS to local


Store File

- upload & store a local file given name to SUFS


Delete File

- delete a file given name from SUFS if it exists


Show Directory

- display the contents of file directory on SUFS


Create Directory

- create a file folder given name & path on SUFS


Delete Directory

- delete a file folder given name & path on SUFS
